# Coding Learning Platform

A complete interactive coding learning platform with student accounts and admin dashboard.

## Features

### 🔐 Student System
- Username = Password registration
- Persistent progress tracking
- Unique usernames only
- XP and leveling system

### 🛡️ Admin Dashboard
- Student monitoring and management
- Progress tracking
- Individual student controls
- Data export functionality

### 🎨 UI/UX
- Dark/Light theme toggle
- Responsive design
- Modern interface
